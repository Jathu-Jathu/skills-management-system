# Skills Management System

A full-stack web application for consultancies and tech agencies to manage personnel skills and match them to project requirements.

## Features

### Core Features
1. **Personnel Management** - CRUD operations for team members
2. **Skill Management** - Create and manage skills catalog
3. **Project Management** - Create projects with required skills
4. **Smart Matching** - Algorithm to find best personnel for projects

### Additional Features
1. **Advanced Search & Filtering** - Search personnel by skills, experience level
2. **Personnel Utilization Dashboard** - Visualize team workload and allocation
3. **Responsive Design** - Works on mobile and desktop
4. **Real-time Validation** - Form validation on frontend and backend

## Technology Stack

### Frontend
- React.js 18.2.0
- React Router DOM 6.15.0
- Axios for API calls
- Chart.js for visualizations
- React Icons

### Backend
- Node.js with Express.js
- MySQL database
- MySQL2 for database connection
- CORS for cross-origin requests

### Database
- MySQL 8.0+

## Prerequisites

1. Node.js (v16 or higher)
2. MySQL (v8.0 or higher)
3. Git
